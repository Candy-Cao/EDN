一个轻量级客户端异步IO网络库；支持链路复用、异步数据收发、定时器、信号处理

本地构建需要linux开发环境
转到EDN/entry/src/main/cpp/src目录；执行
```
make build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
make

```
